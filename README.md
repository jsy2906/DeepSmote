# DeepSmote
DeepSmote는 데이터 불균형을 완화시키기 위한 코드입니다.  
SMOTE 기법에 딥러닝을 접목시킨 모델로, 데이터를 생성하지만 GAN 모델과는 다르게 discriminator로 예측하는 부분을 필요치 않습니다.  
AutoEncoder 모델을 활용해 더 손쉽게 데이터를 oversampling 시킬 수 있습니다.  

(코드는 계속 수정중입니다)
